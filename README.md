## 🛠️ 開發進度更新

### 2026-04-09 專案架構大改版 (Modulization)
為了提升開發效率與程式碼可讀性，今天完成了核心邏輯的拆分與優化：

- **[新] config.js**: 集中管理磚塊數值、顏色配置與遊戲常數。
- **[新] renderer.js**: 抽離渲染引擎，專門處理繪製特效與像素風格魔法棒。
- **[新] physics.js**: 獨立物理碰撞演算法，解決穿模問題並優化得分判定。
- **[優化] sound.js**: 修復 Web Audio API 邏輯，恢復「降階俯衝」大招音效，並支持 `BGM/` 資料夾路徑。
- **[優化] skills.js**: 整合主動大絕與預留被動魔法技能系統。
- **[優化] game.js**: 瘦身成功！轉型為純粹的狀態指揮官。
